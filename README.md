Proxy InvestC
===================

## Synopsis

Proxy InvesTc to manage InvesTc project.

## Installation

``` bash
# Clone the project
git clone git@github.com:Jngueno/proxy-investc.git
cd proxy-investc

# Set environment variables
make env

# Install and mount the environment
make
```

## Usage

``` bash
#Run and watch
make watch

#Debug app
make debug

#Display logs
make logs

#Run container
make run

#Stop container
make stop

#Restart container (after changes)
make restart

#Enter in docker bash
make bash

#Run Eslint
make lint

#Fix Eslint
make fix-lint
```
